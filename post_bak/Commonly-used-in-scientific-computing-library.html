NumPy

NumPy (Numeric Python)：用 Python 实现的科学计算库

  1. 强大的 N 维数组对象 array
  2. 成熟的科学函数库
  3. 实用的线性代数、随机数生成函数等
  4. NumPy 的操作对象是多维数组 ndarray
  • ndarray.shape 数组的维度
  • 创建数组：np.array()，np.arrange() …
  • 改变数组形状 reshape()
