移动零

给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

示例:

说明:

  1. 必须在原数组上操作，不能拷贝额外的数组。
  2. 尽量减少操作次数。

解法

一维数组的坐标变换

  • j 指针指向零元素的位置，只要是非零就移动指针指向下一个，如果是零就停下来不再移动，然后 i 指针继续向后走，遇到非零元素，交换两者位置，实现了把 0 向后移动
